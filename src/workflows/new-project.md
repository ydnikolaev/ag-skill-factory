# New Project Workflow

Start discovery for a brand new project.

## Steps

1. Activate `@idea-interview` skill
2. Conduct structured interview (Elevator Pitch → JTBD → Features)
3. Create `discovery-brief.md` artifact
4. On approval → auto-delegate to `@product-analyst`

## Trigger

- "I have a new project idea"
- "Let's start a new project"
- User provides initial concept
