# Section Template

Use this format when creating new skill sections for `.agent/templates/skill sections/`.

```markdown
## <Section Name>

> <One-line description: what this section enforces or enables>

See [<RULE_FILE>.md](file://.agent/rules/<RULE_FILE>.md) for full rules.
```

## Example

```markdown
## Language Requirements

> Ephemeral artifacts in user's language; persistent docs in English only.

See [LANGUAGE_REQUIREMENTS.md](file://.agent/rules/LANGUAGE_REQUIREMENTS.md) for full rules.
```
