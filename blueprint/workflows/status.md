# Status Workflow

Get instant project status summary.

## Steps

1. Activate `@project-bro` skill
2. Read `ARTIFACT_REGISTRY.md`
3. Read active documents
4. Provide status summary:
   - Current phase
   - Blockers
   - Next steps
   - Suggested skill activation

## Output

Conversational response (no persistent artifact).

## Trigger

- "Bro, where are we?"
- "Project status"
- "What's the current state?"
