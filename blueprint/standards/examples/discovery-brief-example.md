# Discovery Brief: Fitness Bot

## Elevator Pitch

A Telegram Mini App that tracks workouts and provides personalized training recommendations, helping users maintain fitness habits with minimal friction.

## Upstream Documents

- User interview notes (conversation artifact)

## Problem Statement

Users struggle to maintain consistent workout habits. Traditional apps are too complex, require separate installation, and feel disconnected from daily messaging flow.

## Proposed Solution

Telegram Mini App with:
- Quick workout logging (one tap for common exercises)
- Weekly progress summaries
- Personalized recommendations based on history

## Primary Users

1. **Casual fitness enthusiasts** — want simple tracking
2. **Gym beginners** — need guidance on routines

## Jobs to Be Done

| When I... | I want to... | So I can... |
|-----------|--------------|-------------|
| Finish a workout | Log it quickly | Track my progress |
| Start my week | See what to do | Stay consistent |
| Feel unmotivated | Get a nudge | Stay on track |

## Core Features

1. ✅ One-tap workout logging
2. ✅ Weekly summary notifications
3. ✅ Basic exercise library
4. ⏳ Personalized recommendations (Phase 2)
5. ⏳ Social sharing (Phase 3)

## Non-Goals

- Detailed nutrition tracking
- Video tutorials
- Wearable integrations (MVP)

## Success Metrics

| Metric | Target |
|--------|--------|
| Weekly active users | 1000 |
| Workout logs per user per week | 3 |
| 30-day retention | 40% |
